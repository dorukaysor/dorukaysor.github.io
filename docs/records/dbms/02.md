### **Aim**
> Create a basic database and perform simple operations (create, insert, update, delete)

### **Queries**
- `CREATE DATABASE`: create a database
```sql
-- Create the database
CREATE DATABASE CollegeDB;
USE CollegeDB;
```

- `CREATE TABLE`: create a table
```sql
-- Create the table
CREATE TABLE Student (
    RollNo INT PRIMARY KEY,
    Name VARCHAR(50),
    Department VARCHAR(20),
    CGPA DECIMAL(3, 2)
);
```

- `INSERT INTO`: Insert data into the Table
```sql
INSERT INTO Student (RollNo, Name, Department, CGPA) VALUES 
(101, 'Amit Sharma', 'CSE', 8.5),
(102, 'Priya Singh', 'ECE', 9.0),
(103, 'Rohan Das', 'ME', 7.8),
(104, 'Sneha Gupta', 'CSE', 8.2);
```

- `SELECT`: View the table
```sql
SELECT * FROM Student;
```

- `UPDATE`: update or modify a record
```sql
UPDATE Student 
SET CGPA = 8.0 
WHERE RollNo = 103;

-- Verify the update
SELECT * FROM Student WHERE RollNo = 103;
```

- `DELETE`: delete data from the table
```sql
DELETE FROM Student 
WHERE RollNo = 104;

-- Verify the deletion
SELECT * FROM Student;
```

### **Conclusion**
> Successfully created a basic database and performed simple operations such as create, insert, update, and delete on the Student table.
