### **Aim**

> To write SQL queries for table creation and modification

### **Query Syntax**

- **CREATE TABLE:**
  ```sql
    CREATE TABLE table_name (
      column1 datatype,
      column2 datatype,
      ...
    );
  ```
- **ALTER TABLE:**
  - **ADD COLUMN:**
    ```sql
        ALTER TABLE table_name
        ADD column_name datatype;
    ```
  - **MODIFY COLUMN:**
    ```sql
        ALTER TABLE table_name
        MODIFY column_name new_datatype;
    ```
  - **DROP COLUMN:**
    ```sql
        ALTER TABLE table_name
        DROP COLUMN column_name;
    ```
- **RENAME TABLE:**
  ```sql
      RENAME TABLE old_table_name TO new_table_name;
  ```
- **TRUNCATE TABLE:**
  ```sql
      TRUNCATE TABLE table_name;
  ```
- **DROP TABLE:**
  ```sql
      DROP TABLE table_name;
  ```

### **Example Script**

```sql
CREATE TABLE Student_Info (
    RollNo INT PRIMARY KEY,
    Name VARCHAR(50),
    Stream VARCHAR(20)
);

ALTER TABLE Student_Info
ADD Mobile VARCHAR(10);

ALTER TABLE Student_Info
MODIFY Stream VARCHAR(50);

ALTER TABLE Student_Info
DROP COLUMN Mobile;

RENAME TABLE Student_Info TO Alumni;

DESCRIBE Alumni;

TRUNCATE TABLE Alumni;

DROP TABLE Alumni;
```
